My Projects

Higher Ground Shopping Website

Built With: React, React Router, Bootstrap, HTML5 & CSS.

Higher Ground is a mock website of a fake outdoor brand. This website sells many products suited for your outdoor needs. You can add, remove & change the quantity of products in your shopping cart. It will add up the total for you so you know how much everything costs. As this is just a mock website, you cannot actually checkout but you can browse all the fun products!

Specialty Coffee Simplified

Built With: React + Vite, React Router, Bootstrap, Prettier, HTML5 & CSS.

Specialty Coffee Simplified is an informative website about a subject I am really interested in: coffee. It has different types of information regarding what each region in the world offers coffee wise and what flavor profiles come along with that. It also has a section explaining different processes used for harvesting coffee alongside what Specialty Coffee is. This website is still in progress so I most likely will be adding more on, but for now, enjoy!

Sketchpad

Built With: Javascript, HTML5 & CSS.

Etch a Sketch is a fun little sketchpad built with the Odin Project. This project was made to practice DOM manipulation. You can choose from either black or a randomized color option to draw with. You can also resize, clear or erase the board if you so choose!

Weather App

Built With: OpenWeatherMap API, Javascript, HTML5 & CSS.

This is a weather app that shows a 7-day forecast for the current city being searched. It uses OpenWeatherMap API to fetch the weather. It gives you an overview of the high's and low's for each day, corresponding with an icon indicating the conditions.
